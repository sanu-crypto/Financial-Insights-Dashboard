# Financial-Insights-DashboardBusiness Financial Performance Dashboard
📌 Project Overview

The Business Financial Performance Dashboard is an interactive analytics dashboard created using Power BI to monitor and analyze an organization’s financial performance.

This dashboard provides insights into:

Revenue
Profit
Expenses
Transactions
Department-wise performance
Daily profit trends

It helps businesses make data-driven decisions by visualizing key financial metrics in an easy-to-understand format.

🚀 Features

✅ KPI Cards for:

Total Revenue
Total Profit
Total Expenses
Total Transactions
Average Revenue
Average Profit
Average Expenses

✅ Department-wise Analysis

Revenue by Department
Expenses by Department
Profit by Department

✅ Interactive Visualizations

Bar Charts
Pie Charts
Scatter Plot
Slicers & Filters

✅ Dynamic Filtering

Filter data by Date
Filter by Department

✅ Professional Dashboard UI Design
